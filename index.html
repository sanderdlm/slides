Features

  • C, dus cross-platform
  • 100% open-source
  • ACID compliant
  • Uitgebreid getest (meer tests dan source code)
  • Stabiel & LTS
  • Small: 600kb
  • Gemakkelijk backup/deploy want 1 file

Limits & myths

  • 1 writer per keer
  • 5 seconden PENDING default
  • Concurrency -> Write ahead log (WAL)
  • 50,000 inserts/s
  • Corrupt -> automatische rollback
  • 140TB - 1GB
